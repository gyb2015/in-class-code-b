Movie Box Office Data for 2012

This service can return various box office data for 2012 in JSON format. To get this data, you can request various URLs:

  • /movies/ will return an array of movie objects
  • /genres/ will return a list of all distinct genres (e.g., 'drama', 'comedy', etc.)
  • /genres/genre-name/ will return overall stats for the particular genre requested
  • /genres/genre-name/movies/ will return all movies for the particular genre
  • /studios/ will return a list of all distinct studios (e.g., 'walt disney', 'focus features', etc)
  • /studios/studio-name/ will return overall stats for the particular studio requested
  • /studios/studio-name/movies/ will return all movies for the particular studio
  • /ratings/ will return a list of all distinct ratings (e.g., 'g', 'pg-13', etc)
  • /ratings/rating-name/ will return overall stats for the particular rating requested
  • /ratings/rating-name/movies/ will return all movies for the particular rating
